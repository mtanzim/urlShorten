FCC Backend Challenges

Challenge 03:URL Shortener Service

User stories:

    1. I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
    2. When I visit that shortened URL, it will redirect me to my original link

Example Links:

https://fcctimestampmicroservice.glitch.me/December%2015,%202015
https://fcctimestampmicroservice.glitch.me/1450137600
https://fcctimestampmicroservice.glitch.me/thisisnotadate

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
